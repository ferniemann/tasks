CLI Git

Git-Cheatsheet

CLI ist die Abkürzung für "Call Level Interface". Das CLI ist die einfachste Möglichkeit der Interaktion mit einem Computer. Die Eingaben und Ausgaben erfolgen in einem sogenannten Terminal.

Diese Seite gibt Informationen zu den grundlegendsten Befehlen für ein CLI. Klicke auf einen Befehl, um mehr darüber zu erfahren und Beispiele anzusehen.

  • pwd
  • ls
  • cd
  • cat
  • mkdir
  • touch
  • mv
  • cp
  • rm

init

Eine Repository initiieren

add

Datei(en) und/oder Verzeichnis(se) vom Arbeitsverzeichnis in die Staging Area kopieren

commit -m "message"

Der Befehl commit -m "message" erstellt einen Snapshot, wobei "message" durch eine beliebige Mitteilung ersetzt werden kann, die anderen Mitarbeitenden Hinweise darauf gibt, was geändert wurde.

log

Alle commits anzeigen, die gemacht wurden

branch

"branch" gefolgt von einem beliebigen gewünschten branch-Titel erstellt eine neue branch

checkout

checkout gefolgt von dem branch-Namen, wechselt in die entsprechende branch.

commit --amend

commit rückgängig machen
